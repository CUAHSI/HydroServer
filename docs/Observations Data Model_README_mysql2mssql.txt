	2. install on your machine
	
	3. run the script

if the mysql database is on linux machine
	1. install either from the source code https://pypi.python.org/packages/source/M/MySQL-python/MySQL-python-1.2.4.zip#md5=ddf2386daf10a97af115ffad2ed4a9a0
	
	2. or otherwise  mysqldb package for python from your package install i.e yum or apt
	
	3. Run the script

In the SQL Server management studio, login and open the sql file. execute.
