CreateValues method test

CreateValues method. enter one or more data values in the xml. The DateTimeUTC attribute and the value are required. The qualifier and censorCode elements are optional.

Use a HTTP POST request with following parameters:

authToken
the authorization key.
siteCode
the site code.
variableCode
the variable code.
methodCode
the method code (optional)
sourceCode
the source code (optional)
qualityControlLevelCode
the quality control level code (optional)
data
the list of data values. The list should be in XML or CSV format.
format
the data format. Default format is XML.
