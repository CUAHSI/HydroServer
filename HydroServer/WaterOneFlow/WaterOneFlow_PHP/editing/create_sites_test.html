CreateSites method test

CreateSites method. enter one or more sites in the xml. The siteName, latitude and longitude elements are required. The elevation_m and siteCode elements are optional.

Use a HTTP POST request with following parameters:

authToken
the authorization key.
data
the list of sites. The list should be in XML or CSV format.
format
the data format. Default format is XML. Optional format is CSV.
